- `moves` is calculated as follows:
  - For the last character in `ch`, the number of moves is `ch[n-1]-'0'`.
  - For each character from index `0` to `n-2`, if the character is greater than '0', then the number of moves is `(ch[i]-'0'+1)`.

   - For `i=2`, `ch[2]` is '7'
