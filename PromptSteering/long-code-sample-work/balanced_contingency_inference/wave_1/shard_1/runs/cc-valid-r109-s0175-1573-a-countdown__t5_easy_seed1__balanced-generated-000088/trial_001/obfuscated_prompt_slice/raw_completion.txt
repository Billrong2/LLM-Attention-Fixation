


### CASE 001: find(n, ch); stdin="7\n3\n007\n4\n0000\n5\n00000\n3\n103\n4\n2020\n9\n189517537\n30\n001678294039710047203946100020"
- `n = 7`
- `ch = "007"`

- `moves = 0`
- Loop `for(int i=0;i<n-1;i++)`:
    - First iteration:
        - `i = 0`
        - `ch[i] = '0'`
        - `ch[i] - '0' = 0`
        - `0 > 0` is false, so no update to `moves`
    - Second iteration:
        - `i = 1`
        - `ch[i] = '0'`
        - `ch[i] - '0' = 0`
        - `0 > 0` is false, so no update to `moves`
- `moves += ch[n-1] - '0'`
    - `moves += '7' - '0' = 7`
    - `moves = 7`
- Return value: `7`
    
The output of this program for `find(7, "007")` is `7`.

The output of this program for `find(4, "0000")` is `0`.
