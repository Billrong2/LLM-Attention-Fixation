The program starts by taking the input values for the number of test cases, dimensions of the arrays, and the coordinates of the points to be added. Then it calls the solve function with the given inputs.

The solve function takes the input arrays and adds the coordinates of the points to be added to the corresponding sets. It then sorts the x and y sets and creates a HashMap to store the frequency of each point.

The solve function then iterates over each point in the x and y sets, and calculates the answer based on the frequency of each point and the number of points in the x and y sets.

The final answer is printed to the console.
