The program reads an integer `n` and a string of characters, then calculates and prints the number of moves needed to sort the characters in the string in ascending order. The number of moves is the number of times the characters need to be swapped to achieve the sorted order.

For the input:

```
7
3
007
4
1100
5
00010
3
103
4
2020
9
123456789
30
001678294039710047203946100020
```

The program will print:

```
2
3
2
2
3
5
26
```
